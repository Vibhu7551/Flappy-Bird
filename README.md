# 🐤 Flappy Bird Clone

A simple **Flappy Bird** clone built using **Python** and **Pygame**.  
Control the bird, avoid the pipes, and try to get the highest score! 🚀

---

## 🎮 How to Play
- Press **Spacebar** to make the bird jump  
- Don’t hit the pipes  
- Don’t fall down  

---

## 🖥️ Requirements
- Python 3.x  
- Pygame
